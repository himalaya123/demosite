This is a database driven dynamic website where a user could Create, Read, Update & Delete (CRUD) posts .

The main application is named as 'mainsite' which consits of code for three pages - HOME, ABOUT and NEWS.

=> HOME page :- This is a simple default home page with a navigation bar.


=> ABOUT page :- This is a default about page .

=> NEWS page :- This is a page where a user could read to other people's posts,update and delete.
